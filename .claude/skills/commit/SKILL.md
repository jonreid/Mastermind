STARTER_CHARACTER = 💾

## Description

Standardized process for creating meaningful git commits using Arlo's notation.

## Steps

1. **Review Changes**

2. **Draft Commit Message**
- Draft a commit message following [Arlo's Commit Notation](~/.claude/skills/commit/references/commit-messages-arlo-belshee.md)
- **ALWAYS** ask the programmer to approve the commit message.

3. **Commit**
